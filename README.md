### About

This is a solution for this problem https://adventofcode.com/2024/day/2 written in node

`sample.txt` is the sample input provided in the problem statement, and `input.txt` is the actual input for the problem provided from https://adventofcode.com/2024/day/2/input.

You can replace `input.txt` with `sample.txt` in `app.js` to test with the sample input.

node ~v24


### Run
